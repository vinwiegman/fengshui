# Roadmap

## Phase 1: Finished POC

- Expo mobile app.
- Photo picker.
- Simulated analysis loading.
- Fake detected-object overlays.
- Editable inventory.
- Detection confidence and zone cards.
- Goal selection.
- Constraint toggles.
- Heuristic analysis and plan generation.
- Share/export summary screen.
- GitHub-ready documentation.

## Phase 2: AI-Assisted MVP

- Add backend endpoint for photo analysis.
- Send uploaded image to a vision-capable model.
- Return structured JSON for furniture, clutter, style, and layout issues.
- Populate the editable inventory from model output.
- Add schema validation for all AI responses.

## Phase 3: Better Spatial Reasoning

- Add multiple room photos.
- Merge duplicate object detections across photos.
- Infer rough spatial relations.
- Score circulation, clutter, lighting, workspace, and access.
- Reject actions that violate hard constraints.

## Phase 4: Visual Planning

- Add annotated arrows on the uploaded photo.
- Add a simple 2D layout board.
- Add concept visualization with clear approximation labels.

## Phase 5: Productization

- Save room projects.
- Add authentication.
- Store images and generated analysis.
- Add sharing/export.
- Add evaluation set and recommendation-quality metrics.
